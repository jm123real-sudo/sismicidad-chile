\# Fase 3 – Núcleo Algorítmico, Eficiencia y POO



\## Proyecto

Análisis de sismicidad en Chile.



\## Objetivo

Desarrollar un núcleo algorítmico modular para analizar eventos sísmicos utilizando programación estructurada, recursividad, programación orientada a objetos y mediciones de eficiencia computacional.



\## Dataset utilizado

Se utiliza el archivo `F1/seismic\_data.csv`, compuesto por registros sísmicos con las variables:



\- Date(UTC)

\- Latitude

\- Longitude

\- Depth

\- Magnitude
